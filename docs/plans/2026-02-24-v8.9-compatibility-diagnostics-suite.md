# v8.9 Compatibility Diagnostics Suite Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a first-class `engram compat` diagnostics suite that verifies OpenClaw hook/plugin/runtime compatibility and surfaces actionable remediation guidance.

**Architecture:** Implement deterministic compatibility checks across plugin manifest, hook registration, binary availability, config schema parity, and runtime assumptions. Provide human and JSON output modes for operators/automation.

**Tech Stack:** TypeScript, `src/commands/compat.ts` (new), `src/cli.ts`, node:test.

---

## Requirements

1. Compatibility report with `ok|warn|error` checks.
2. JSON output mode for CI/ops tooling.
3. Strict mode exits non-zero on warnings and errors.
4. Include remediation hints per failed check.
5. No network dependency for baseline checks.
