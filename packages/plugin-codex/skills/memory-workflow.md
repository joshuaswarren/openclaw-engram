Engram memory commands for Codex:

- **Store a memory**: Use the `engram_memory_store` MCP tool to save information
- **Recall memories**: Use the `engram_recall` MCP tool with a natural language query
- **Search**: Use `engram_lcm_search` for deep full-text search
- **Entities**: Use `engram_entity_get` to browse the knowledge graph
- **Status**: Run `engram daemon status` via shell to check system health

All memories are shared across connected agents (Claude Code, OpenClaw, Hermes, etc.).
